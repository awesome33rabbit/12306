# 12306
Buying train tickets
 >https://www.12306.cn/index/

# 特性
* selenium + Chrome 模拟登陆 12306
* 购票查询
* 购票预订
* 添加购票乘客
* 提交订单

# 使用
使用 Python3.6 开发的项目，没有对其他版本 Python 测试

## 构建环境
`pip install -r requirements.txt`

## 运行脚本
`selenium_12306.py`
